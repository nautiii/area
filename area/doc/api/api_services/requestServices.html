requestServices function

Future<List<Service>> requestServices ()

API call to get the list of all available services If the request is successful, return the parsed json response from the server as a List of Service If not, return null

Implementation

Future<List<Service>> requestServices() async {
  final response = await http.get(
    Uri.parse('http://${conf.ip}:${conf.port}/api/actions'),
    headers: <String, String>{
      'Content-Type': 'application/json; charset=UTF-8',
      'access-token': api.token,
    },
  );

  if (jsonDecode(response.body)['success'] == false) return null;
  return (jsonDecode(response.body)['services'] as List)
      .map((value) => Service.fromJson(value))
      .toList();
}
